## 게임 명세서
### 게임 컨셉

1. 만든 빵으로 사람을 강화시킬 수 있다 → 보유 중인 레시피 중 랜덤으로 3개씩, 일정 시간마다
2. 제한된 수의 포탑을 키울 수 있음 
3. 특정 몬스터를 잡으면 특별한 템을 떨군다 


### 접속 방법

1. 로그인 기능 제공 → 신규 유저면 닉네임 받음

### **로비**

1. **게임 시작 방식**
    1. 방 생성 및 방 참가
    2. 개인 플레이 페이지가 주 + 멀티플레이는 부
       
    
2. **설정(우선순위 순)**
    1. **사운드**
        1. bgm
        2. 효과음
        3. 마스터 볼륨
    2. **키 매핑(세팅)**
    3. 디스코드 링크
    4. 이펙트 관련 on/off
    5. ~~해상도~~(하나로 고정해서 사용)

### 방 설정

1. 맵 설정
2. 방 이름 바꾸기
3. 암호 설정
4. 인원 설정
    1. 최대 4명
5. 강퇴 기능(방장만)
